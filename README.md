# E-Learning Platform Landing Page & Dashboard

## Overview
A comprehensive e-learning platform featuring an animated landing page and an interactive dashboard with data visualizations. The platform helps users track their learning progress, course completion rates, and study hours through intuitive visualizations.

## Live Demo
Visit the live site: [E-Learning Platform](https://e-learning-ehshan.netlify.app/)

## Author
Mohammad Ehshan

## Features
### Landing Page
- Custom animations to enhance user engagement
- Modern and intuitive user interface
- Interactive elements and smooth transitions

### Dashboard Analytics
- **Course Completion Tracking (Pie Chart)**
  - Visual representation of completed vs pending courses
  - Percentage breakdown of course completion status
  - Interactive segments for detailed course information

- **Learning Progress Visualization (Line Chart)**
  - Tracks student performance percentages over time
  - Progress monitoring across different courses
  - Trend analysis of learning outcomes

- **Study Hours Tracker (Flow Chart)**
  - Detailed breakdown of hours spent on each subject
  - Integrated calendar view for time management
  - Visual representation of study patterns and time allocation

- **Interactive Calendar Integration**
  - Syncs with study hour tracking
  - Schedule visualization
  - Progress tracking by date

## Setup Instructions
1. Clone the repository to your local machine
2. Ensure Chart.js is properly linked in your dashboard HTML file
3. Open `index.html` for the landing page
4. Navigate to dashboard.html for the analytics dashboard
5. No additional setup or dependencies required

## Current Limitations
- The landing page is currently not responsive
- Best viewed on desktop devices
- Charts may require performance optimization for large datasets
