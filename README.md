# E-Learning Platform Landing Page

## Overview
A dynamic landing page for an e-learning platform featuring animated elements. The page is built using HTML and CSS, showcasing modern design principles and engaging user interactions.

## Live Demo
Visit the live site: [E-Learning Platform](https://e-learning-ehshan.netlify.app/)

## Author
Mohammad Ehshan

## Features
- Custom animations to enhance user engagement
- Modern and intuitive user interface
- Pure HTML and CSS implementation
- Interactive elements and smooth transitions

## Technologies Used
- HTML5
- CSS3
  - CSS Animations
  - Flexbox/Grid layouts

## Setup Instructions
1. Clone the repository to your local machine
2. Open `index.html` in a modern web browser
3. No additional setup or dependencies required
